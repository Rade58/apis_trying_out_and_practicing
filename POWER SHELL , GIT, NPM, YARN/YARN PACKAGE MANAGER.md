AKO ZELIM DA SE UPGRADE-UJE NA ONU VERZIJU SPECIFICIRANU U package.json

- yarn upgrade

AKO ZELIM DA SE UPGRADE-UJU NA LATEST VERZIJU (CIME SE MENJAJU SPECIFICIRANE VERZIJE U package.json FAJLU)

- yarn upgrade --latest
