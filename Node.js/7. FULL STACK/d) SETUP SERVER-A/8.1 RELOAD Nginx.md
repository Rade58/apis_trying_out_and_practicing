**EDITOVAO SI KONFIGURACIJU, RED JE I DA RELOADUJES NGINX SERVICE, DA BI SE PROMENE PALICIRALE**

- sudo service nginx reload

**AKO NISI PRAVILNO DEFINISAO NGINX KONFIGURACIJU**, RUNNUJ OVO

- sudo nginx -t
